# Quickstart Guide

Get up and running with BevorAI in under 10 minutes. This guide will walk you through the fastest path to start securing your smart contracts and protocols.

## Step 1: Create Your Account

1. Visit [app.bevor.io](https://app.bevor.io)
2. Sign up with your email or connect your wallet
3. Verify your account and complete onboarding

## Step 2: Get Your API Key

1. Navigate to your dashboard
2. Go to **Settings** â†’ **API Keys**
3. Generate a new API key
4. Copy and securely store your key

<Warning>
  Keep your API key secure and never commit it to version control. Use environment variables or secure secret management.
</Warning>

## Step 4: Run Your First Security Scan

### Using the Web Terminal

1. Go to [app.bevor.io](https://app.bevor.io)
2. Open the Terminal interface
3. Upload or paste your smart contract code
4. Run the security analysis command
5. Review the detailed findings and recommendations

### Using MCP in Your IDE

1. Install the BevorAI MCP extension
2. Configure your API key in the extension settings
3. Open a smart contract file
4. Use the security analysis command palette
5. Review inline security suggestions
